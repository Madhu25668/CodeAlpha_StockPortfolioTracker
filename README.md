# 📈 Stock Portfolio Tracker

A beginner-friendly Python project developed as part of the **CodeAlpha Python Programming Internship**. This application allows users to track their stock investments by entering stock names and quantities. It calculates the total portfolio value using predefined stock prices and saves the results to a text file.

---

## 📌 Project Overview

The Stock Portfolio Tracker is a simple console-based Python application that helps users calculate the value of their stock portfolio. It demonstrates the use of dictionaries, loops, conditional statements, user input, and file handling in Python.

---

## ✨ Features

- Predefined stock price database
- User-friendly console interface
- Accepts multiple stock entries
- Calculates individual stock values
- Displays the total investment value
- Saves portfolio details to a text file
- Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python 3
- Dictionaries
- Loops
- Conditional Statements
- File Handling

## 🚀 How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python stock_tracker.py
```
